# 🦅 TradeReview AI 

## 📖 关于本项目 (Origin Story)

👋 **你好！作者本人是一个完全不懂代码的交易员（Code Noob）。**

这个项目的所有代码（是的，每一行），都是我通过与大语言模型（LLM）对话生成的。我负责提供**交易逻辑、风控模型和复盘痛点**，AI 负责将它们翻译成 Python 代码。

这不是一个为了“秀技术”而生的软件，而是一个**为了“解决交易顽疾”而生的实战工具**。它没有复杂的架构，但每一个功能都直击交易员的灵魂：**FOMO、卖飞、扛单、知行合一。**

没错就是纯粹自己使用，没错这个简介也是AI生成了，我感觉它对功能的总结的不对，但是不修改了。

## ✨ 核心功能 (Key Features)

### 1\. 🦅 上帝视角复盘 (God View Review)

  * **卖飞模拟器**：平仓后价格起飞了？系统会自动推演“如果多拿 4 小时/12 小时”会多赚多少钱，用数据治愈你的“卖飞恐惧症”或“过度贪婪”。
  * **Vegas 趋势审计**：自动拉取大周期 K 线，判断你的这笔交易是顺势还是逆势（基于 144/169/288 均线系统）。

### 2\. 🧠 AI 导师 (AI Coach)

  * **内置 LLM 审计**：支持 DeepSeek、OpenAI、Gemini。
  * **风格**：它不是那种温温柔柔的客服，而是**严厉的风控总监**。它会根据你的《交易宪法》检查你是否违规，并在你亏损时狠狠地批评你的入场逻辑。
  * **RAG 记忆系统**：内置向量数据库，AI 会“记住”你过去的错误。当你再次想追涨时，它会把以前类似的惨痛教训甩在你脸上。

### 3\. 🔬 深度价格行为分析 (Deep PA Stats)

  * **MAE/MFE/ETD**：不只是看赚赔，还要看过程。计算最大浮亏（MAE）和最大浮盈（MFE），评估你的入场效率。
  * **RVOL & ATR**：自动计算入场时的相对成交量和波动率，拒绝死水行舟。
  * **痛苦路径可视化**：在 K 线图上标记出你持仓期间的“痛苦时刻”（浮亏时段）。

### 4\. 🛡️ 蒙特卡洛风控沙盘 (Risk Sandbox)

  * 基于你历史的胜率和盈亏比，模拟未来 1000 个平行宇宙的资金曲线。
  * 计算 **破产概率 (Risk of Ruin)** 和 **最大回撤预期**，在你上头想梭哈时给你一盆冷水。

### 5\. 🏠 数据完全本地化 (Local First)

  * **安全**：API Key、交易记录、复盘笔记全部存储在本地 SQLite 数据库，**绝不上传云端**。
  * **本地 K 线仓库**：内置数据引擎，将交易所 K 线下载到本地，实现毫秒级回放，无需频繁请求 API。

## 🛠️ 技术栈 (Tech Stack)

虽然我是小白，但 AI 帮我选了这些好用的库：

  * **Frontend**: `Streamlit` (极简的 Python Web 框架)
  * **Data**: `Pandas`, `SQLite`, `CCXT` (交易所接口)
  * **Analysis**: `Pandas-TA` (技术指标), `Plotly` (交互式图表)
  * **AI/RAG**: `OpenAI SDK`, `ChromaDB` (向量数据库)
  * **Deploy**: `Docker`

## 🚀 快速开始 (Quick Start)

### 方式一：Docker 一键部署 (推荐)

最省心的方式，不需要配置 Python 环境。

```bash
# 1. 克隆仓库
git clone https://github.com/your-username/TradeReview-AI.git
cd TradeReview-AI

# 2. 运行脚本 (Mac/Linux)
chmod +x docker-run.sh
./docker-run.sh

# 2. 运行脚本 (Windows PowerShell)
./docker-run.ps1
```

启动后访问：`http://localhost:8501`

### 方式二：本地 Python 运行

```bash
# 安装依赖
pip install -r requirements.txt

# 初始化 TA-Lib (如果遇到报错，请自行搜索如何安装 TA-Lib 的 C++ 库)
# Windows 用户建议直接下载 whl 文件安装

# 启动应用
streamlit run app.py
```
